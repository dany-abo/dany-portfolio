## 🎨 التصميم

### الألوان
- [ ] الألوان متناسقة ومناسبة للمجال
- [ ] التباين جيد وسهل القراءة
- [ ] الألوان تعمل في Dark Mode (إن وجد)
- [ ] لا توجد ألوان متعارضة أو مزعجة

### الخطوط
- [ ] الخطوط واضحة وقابلة للقراءة
- [ ] حجم الخط مناسب (16px على الأقل للنص الأساسي)
- [ ] الخط العربي يعمل بشكل صحيح
- [ ] الخط الإنجليزي يعمل بشكل صحيح (عند التبديل)
- [ ] التباعد بين الأسطر مريح

### الصور

- [ ] الصور عالية الجودة (ليست مشوشة)
- [ ] الصور محسّنة (حجم ملف معقول)
- [ ] الصور لها نص بديل (alt text)
- [ ] الصورة الشخصية احترافية
- [ ] صور المشاريع واضحة وجذابة

### المسافات والتنسيق
- [ ] المسافات البيضاء مريحة للعين
- [ ] العناصر متناسقة ومنظمة
- [ ] لا يوجد ازدحام في التصميم
- [ ] الأقسام واضحة ومفصولة جيداً

### الرسوم المتحركة
- [ ] الرسوم المتحركة سلسة وليست مزعجة
- [ ] لا تؤثر على الأداء

## 🔍 SEO (تحسين محركات البحث)

### Meta Tags الأساسية
- [ ] Title محسّن (50-60 حرف)
- [ ] Description محسّن (150-160 حرف)
- [ ] Keywords مناسبة
- [ ] Author محدد
- [ ] Viewport محدد

### Open Graph (Facebook)
- [ ] og:title محدد
- [ ] og:description محدد
- [ ] og:image محدد (1200x630px)
- [ ] og:url محدد
- [ ] og:type محدد

### Twitter Card
- [ ] twitter:card محدد
- [ ] twitter:title محدد
- [ ] twitter:description محدد
- [ ] twitter:image محدد

### Schema Markup
- [ ] Person schema موجود
- [ ] البيانات صحيحة ومحدثة
- [ ] تم اختباره على Schema Validator

### هيكلية HTML
- [ ] استخدام صحيح لـ H1, H2, H3
- [ ] H1 واحد فقط في الصفحة
- [ ] العناوين منطقية ومتسلسلة
- [ ] استخدام semantic HTML (header, nav, main, section, footer)

### الروابط والصور
- [ ] جميع الصور لها alt text وصفي
- [ ] الروابط الخارجية لها target="_blank"
- [ ] الروابط الخارجية لها rel="noopener"
- [ ] لا توجد روابط مكسورة

### ملفات إضافية
- [ ] robots.txt موجود
- [ ] sitemap.xml موجود
- [ ] favicon موجود
