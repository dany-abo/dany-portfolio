## 📝 What You Can Edit:

### Easy Edits (No coding needed):
1. **Replace placeholder image:**
   - Find: `https://via.placeholder.com/400x400`
   - Replace with your photo URL

2. **Update projects:**
   - Edit project details in `script.js`
   - Look for `projectDetails` object

3. **Change colors:**
   - Open `styles.css`
   - Find `:root` section
   - Change `--primary-color` and `--secondary-color`

## 🎨 Customization Tips:

### To add your real photo:
1. Save your photo as `profile.jpg`
2. Create `images/` folder
3. Put photo inside
4. Update HTML: `src="images/profile.jpg"`

### To add project images:
1. Save project screenshots in `images/projects/`
2. Update image URLs in HTML
3. Recommended size: 400x300px

### To change experience:
1. Open `index.html`
2. Find `<!-- Experience Section -->`
3. Edit the timeline items with your real experience
